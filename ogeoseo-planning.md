## 1. 프로젝트 개요

### 1.1 배경

성균관대학교 학술정보관의 기존 "오거서" 프로그램을 디지털 플랫폼으로 확장한다. 단순 독후감 작성을 넘어, **읽기·듣기·쓰기**를 아우르는 5개의 참여형 존(Zone)을 통해 학생들의 독서 경험을 확장하고, 게이미피케이션을 통해 지속적 참여를 유도한다.

### 1.2 페르소나

> "평범함은 거부한다."

오거서 플랫폼은 전통적인 독서 프로그램의 틀을 깨고, 음악·필사·낭독·커뮤니티 등 다양한 감각적 경험을 독서와 연결하는 것을 목표로 한다.

### 1.3 핵심 기능 요약

| Zone | 핵심 활동 | 카테고리 |
|------|---------|---------|
| 북클럽 존 | 성균인 명사와 함께하는 월간 북클럽 | 읽기 |
| 낭독 존 | 이용자 참여형 오디오북 제작 | 듣기 |
| 듣기 존 | 노래 감정 분석 → 도서 구절 매칭 | 듣기 |
| 필사 존 | 마음에 드는 구절 필사 및 공유 | 쓰기 |
| 서평 존 | 오거서 북리스트 도서 서평 작성 | 쓰기 |

## 2. 사용자 역할 및 권한

### 2.1 역할 정의

| 역할 | 설명 | 권한 |
|------|------|------|
| **학생 (Student)** | 성균관대 재학생. 킹고 ID로 로그인 | 존 참여, 콘텐츠 열람/생성, 포인트 적립, 오거서몰 이용 |
| **관리자 (Admin/MD)** | 학술정보관 사서 | 존별 콘텐츠 관리, 회원 관리, 포인트/배지 운영, 통계 열람, 오거서몰 상품 관리 |
| **좌장 (Moderator)** | 북클럽 존의 명사 좌장 | 해당 북클럽 게시판 관리, 토론 진행 |

### 2.2 인증 방식

- **학생**: 성균관대 통합 로그인(SSO) 연동 (샘플에서는 이메일/비밀번호 방식으로 모의)
- **관리자**: 별도 관리자 계정 (역할 기반 접근 제어)
- **좌장**: 관리자가 생성한 초대 링크 통해 가입

## 4. 게이미피케이션 시스템

### 4.1 포인트 시스템

#### 포인트 적립 규칙

| 활동 | 포인트 | 난이도 | 비고 |
|------|--------|--------|------|
| 북클럽 존 — 토론 참여 (1회) | 30P | ★★★ | 월 1회 참여 기준 |
| 북클럽 존 — 후기 작성 | 20P | ★★ | |
| 낭독 존 — 낭독 녹음 제출 | 25P | ★★★ | 월 1회 |
| 낭독 존 — 오디오북 청취 완료 | 10P | ★ | |
| 듣기 존 — 북 플레이리스트 생성 | 15P | ★★ | 노래-구절 매칭 1건 |
| 필사 존 — 필사 업로드 | 20P | ★★ | |
| 필사 존 — 좋아요 10개 달성 | 10P (보너스) | - | 자동 지급 |
| 서평 존 — 서평 작성 | 25P | ★★★ | 300자 이상 |

#### 포인트 유형

| 유형 | 설명 | 용도 |
|------|------|------|
| **누적 포인트** | 전체 활동으로 획득한 총 포인트 (차감 불가) | 랭킹 경쟁, 장학금 선정 기준 |
| **가용 포인트** | 현재 사용 가능한 포인트 (오거서 몰 사용 시 차감) | 오거서 몰 상품 교환 |

### 4.2 배지 시스템

| 배지 | 조건 | 아이콘 컨셉 |
|------|------|-----------|
| 📖 북클럽 배지 | 북클럽 존 1회 이상 참여 완료 | 책과 대화방울 |
| 🎙️ 낭독 배지 | 낭독 존 1회 이상 녹음 제출 | 마이크와 음파 |
| 🎵 듣기 배지 | 듣기 존 1회 이상 매칭 완료 | 헤드폰과 책 |
| ✍️ 필사 배지 | 필사 존 1회 이상 업로드 | 펜과 종이 |
| 📝 서평 배지 | 서평 존 1회 이상 서평 작성 | 별과 연필 |
| 🏆 문호(文豪) 칭호 | 5개 배지 모두 획득 | 왕관과 책 |

**문호 달성 보상**: 오거서 키링 제공 + 프로필에 "문호" 칭호 표시

### 4.3 경쟁 시스템

#### 개인 랭킹
- **기준**: 누적 포인트
- **표시**: 전체 랭킹 + 이번 달 랭킹
- **보상**: 학기 종료 시 상위 n명 장학금

#### LC(팀) 랭킹
- **기준**: LC 소속 학생들의 누적 포인트 합산
- **목적**: 1학년 참여 유도 (LC는 1학년 소속 팀)
- **표시**: LC별 총 포인트 + 참여율

### 4.4 장학금 규칙

- **자격 조건**: 5개 배지 모두 획득 (문호 칭호 보유자)
- **선정 기준**: 자격 충족자 중 누적 포인트 상위 n명
- **지급 시기**: 학기 종료 시
- **n값**: 관리자가 학기별로 설정

## 부록: 유저 플로우 다이어그램

### A. 신규 사용자 온보딩

```
로그인/가입 → 메인 홈 → 5개 존 탐색 → 관심 존 참여
                                           ↓
                                    포인트 적립 → 배지 획득
                                           ↓
                                    오거서 몰 교환 / 랭킹 확인
                                           ↓
                                    5배지 달성 → 문호 칭호 + 장학금 대상
```

### B. 듣기 존 상세 플로우

```
듣기 존 진입
    ↓
유튜브에서 노래 검색 & 감상
    ↓
노래 제목 + 마음에 드는 가사 입력
    ↓
[AI 분석] 가사 감정 분석 (감정 십진 분류)
    ↓
감정 분석 결과 확인 (감정 키워드, 분류)
    ↓
매칭된 도서 구절 추천 목록 확인
    ↓
┌─────────────┬──────────────┬───────────────┐
│ 소장 도서     │ 미소장 도서    │ 저장           │
│ → 대출 링크   │ → 구매 신청   │ → 플레이리스트 │
└─────────────┴──────────────┴───────────────┘
```

### C. 관리자 월간 운영 사이클

```
월초:
├── 이달의 북클럽 생성 (좌장 지정, 도서 선정)
├── 이달의 낭독 도서/구간 선정
└── 듣기 존 감정-도서 매핑 업데이트

월중:
├── 제출물 검수 (낭독, 필사, 서평)
├── 참여 현황 모니터링
└── 포인트 자동 지급 확인

월말:
├── 낭독 오디오북 편집/출판
├── 이달의 필사 배너 선정
├── 월간 통계 리뷰
└── 다음 달 콘텐츠 준비
```
